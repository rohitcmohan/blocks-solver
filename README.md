# blocks-solver

Solves these [sliding block puzzles](https://en.wikipedia.org/wiki/Klotski). Written in Python 3.

~~~
                                                     +-++----++-+
                                                     | ||X  X|| |
                                                     | ||    || |
                                                     | ||    || |
                                                     | ||X  X|| |
                                                     +-++----++-+
                                                     +-++----++-+
                                                     | ||    || |
                                                     | |+----+| |
                                                     | |+-++-+| |
                                                     | || || || |
                                                     +-++-++-++-+
                                                     +-+      +-+
                                                     | | _  _ | |
                                                     +-+      +-+
~~~
