# blocks-solver

Solves these sliding blocks puzzles.

~~~
                            +----++----+
                            |X  X||X  X|
                            +----+|    |
                            +----+|    |
                            |X  X||X  X|
                            +----++----+
                            +----++-+
                            |X  X||X| _
                            +----++-+
                            +-++----+
                            |X||X  X| _
                            +-++----+
                            +-++-++----+
                            |X||X||X  X|
                            +-++-++----+
~~~
